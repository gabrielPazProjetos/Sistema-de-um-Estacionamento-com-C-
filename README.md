--- Sistema de Estacionamento - DIO .NET Fundamentos
Este projeto é um sistema de gerenciamento de estacionamento desenvolvido como desafio da trilha .NET da DIO. Ele permite cadastrar, listar e remover veículos, calculando o valor total a ser pago com base no tempo estacionado.

--- Funcionalidades implementadas
Cadastro de veículos (placa)
Listagem de veículos estacionados
Remoção de veículos com cálculo de valor: valor = precoInicial + (precoPorHora * horas)
Menu interativo no console
Validação de entradas e mensagens amigáveis

--- Tecnologias utilizadas
C# com .NET
Projeto console (Microsoft.NET.Sdk)
Organização em classes (Estacionamento.cs)
